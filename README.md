# Flux

このChrome拡張機能は、任意の事前登録したwebサイトのアクセスをブロックするツールです。  
現在は **β版** として公開しています。

- フィードバックは Issues から歓迎します。
- 動作確認は、Google Chrome でのみ行なっています。
- 利用の責任は利用者本人にあることとします。

---

## インストール方法（Chrome）

1. このページの右側のサイドバーから **Releases** （v0.9.0-beta(authentic)と書かれているリンク）ページを開く。
2. flux.zip という ZIP ファイルをダウンロードする。
3. ZIP を解凍する。
4. Chrome で `chrome://extensions` を開く
5. 右上の「デベロッパーモード」を ON にする
6. 「パッケージ化されていない拡張機能を読み込む」をクリック
7. 解凍したフォルダを選択

これで拡張機能が追加されます。

---

## 使い方（簡単）
- ブラウザ右上の拡張機能アイコンをクリックする。
- 設定ボタンから、設定をしてください。
- ピンで固定すると便利です。
